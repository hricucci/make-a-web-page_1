Lesson 1

What is HTML?

HTML (Hypertext Markup Language), is the languange the browsers can understand and show us somethng we won't. This text for example, was wrote with this languange, where you can just type something or put images, buttons, and personalize them.

How does it work?

HTML works with Tag and Elements. For example.
I can make a word becomes Italic, by using a tag the letter i inside < and >. the element inside these Symbols define the tag, and these tags make the page look like what we want. If i want to jump line in HTML, i cannot just press enter, i need to put a tag that allows the text be be shown in next line.

And everything follows the same concepts.

Lesson 2

Structure of HTML document

In HTML the layout of the page must be all setted, because the computer don't recognize the formatation of the text by itself. You can do it basically in two different ways: It can be setted as a linear thing or a block. The block "creates an invisible box" around the image or text you want, making it stays where you want.

"Parents and children"

One important and relevant aspect of HTML is that you can put a parameter inside another. By doing this we can set things as a tree, where we choose who is going inside of who. For example we can put a list inside another list, organizing the page as we want.

Text Editors

Every tag must me setted exactly as it should be or the page won't be as we want. For that reason when we write the codes it's very usefull to use some Text Editor for programers thar hepls a lot writing the code. They show options for each tag, and in the most of them, there are colors that describe each thing, making a lot easier to see where there is a mistake.

Lesson 3

Easier and more efficient work

When building a brand new page, there are lots of codes to write, and most of them we may be used several times. And then there is an efficient tool to help in this journey.
We can use an CSS file for almost all the styles we want in the page. By using CSS we can write some style code and use it as many times as we want to. It helps by avoind wiriting the same thing several times and by avoiding error when repeating the same code.

CSS

CSS or Cascading Style Sheets, is basically another document where we write the layout of the HTML. That way we can for example, make a layout for tittles and just by referencing to it, use as many times we want.
